# Developer Profile

I’m a South Africa-based **full-stack web and mobile developer** building practical, polished, and performance-minded digital products. My work spans **frontend interfaces**, **backend systems**, **mobile experiences**, automation, **databases**, **APIs**, and product experiments.

**The goal is simple** : build software that looks professional, feels smooth, and solves real problems without unnecessary complexity.

<br>

| Focus Area | What I Build |
|---|---|
| **Web Apps** | Responsive React / Next.js interfaces, dashboards, landing pages, and portfolio systems. |
| **Mobile Apps** | React Native / Expo apps with practical flows, cloud data, and clean UI structure. |
| **Backend Systems** | Node.js, Express APIs, mail services, auth flows, database-backed tools, and integrations. |
| **UI Systems** | Premium dark interfaces, reusable components, theme systems, motion, and visual polish. |
| **Deployment** | Vercel, Docker, Cloudflare Workers, Caddy, DNS, and lightweight VM-based hosting. |
